# 前后端集成测试指南

## 🚀 启动服务

### 1. 启动后端 API

```bash
# 方式1: 使用启动脚本
python start_backend.py

# 方式2: 直接使用 uvicorn
uvicorn openclaw_studio.api.main:app --reload --host 0.0.0.0 --port 8000
```

后端服务将在 `http://localhost:8000` 启动
- API 文档: http://localhost:8000/docs
- 健康检查: http://localhost:8000/health

### 2. 启动前端开发服务器

```bash
cd openclaw-studio-frontend
npm run dev
```

前端服务将在 `http://localhost:5173` 启动（Vite 默认端口）

## 🔍 错误排查

### 后端问题

1. **端口被占用**
   ```bash
   # Windows
   netstat -ano | findstr :8000
   # 找到进程 ID，然后结束进程
   taskkill /PID <进程ID> /F
   ```

2. **数据库错误**
   - 检查 SQLite 数据库文件是否存在
   - 检查数据库连接配置

3. **API Key 错误**
   - 检查环境变量是否设置
   - 确认 API Key 是否有效

### 前端问题

1. **端口被占用**
   ```bash
   # Vite 会自动尝试下一个端口
   # 或手动指定端口
   npm run dev -- --port 5174
   ```

2. **API 连接失败**
   - 检查 `.env` 文件中的 `VITE_API_BASE_URL`
   - 确认后端服务正在运行
   - 检查浏览器控制台的错误信息

3. **CORS 错误**
   - 确认后端 CORS 配置正确
   - 检查 `openclaw_studio/api/main.py` 中的 CORS 设置
