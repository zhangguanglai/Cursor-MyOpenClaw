# 前端功能测试指南

## 🚀 启动服务

### 1. 确保后端服务运行

```bash
# 如果后端未运行，启动它
python start_backend.py
```

后端服务地址: http://localhost:8000

### 2. 启动前端服务

```bash
cd openclaw-studio-frontend

# 如果 node_modules 不存在，先安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端服务地址: http://localhost:5173

## 🐛 常见问题排查

### 1. 前端无法连接后端
- **问题**: 前端请求失败，显示网络错误
- **解决**: 
  - 确认后端服务运行在 http://localhost:8000
  - 检查 `apiClient.ts` 中的 `baseURL` 配置
  - 检查浏览器控制台的错误信息

### 2. 数据不显示
- **问题**: 页面加载但数据为空
- **解决**:
  - 检查浏览器控制台的网络请求
  - 确认 API 返回数据格式正确
  - 检查 React Query 的缓存状态

### 3. 路由跳转失败
- **问题**: 点击链接后页面不跳转
- **解决**:
  - 检查 `App.tsx` 中的路由配置
  - 确认 `caseId` 参数正确传递
  - 检查浏览器控制台的错误信息

### 4. Markdown 渲染异常
- **问题**: Markdown 内容显示为纯文本
- **解决**:
  - 确认 `react-markdown` 和 `remark-gfm` 已安装
  - 检查组件中的插件配置
  - 确认样式文件已导入

### 5. Diff 预览不显示
- **问题**: Diff Modal 打开但内容不显示
- **解决**:
  - 确认 `react-diff-view` 已安装
  - 检查补丁内容格式是否正确
  - 查看浏览器控制台的错误信息
