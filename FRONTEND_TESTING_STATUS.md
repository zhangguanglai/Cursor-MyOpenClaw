## 🐛 如果遇到问题

### 前端无法访问
1. 确认前端服务正在运行（检查终端输出）
2. 确认端口 5173 未被占用
3. 尝试刷新页面（Ctrl+F5）

### 数据不显示
1. 确认后端服务运行在 http://localhost:8000
2. 检查浏览器控制台的网络请求（F12 → Network）
3. 确认 API 返回数据格式正确

### 页面空白
1. 检查浏览器控制台的错误信息（F12 → Console）
2. 确认所有依赖已安装（`npm install`）
3. 检查路由配置是否正确
