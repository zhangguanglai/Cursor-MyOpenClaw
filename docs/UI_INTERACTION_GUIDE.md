# UI 交互指引

## 🎯 界面概览

### 主界面结构

```
┌─────────────────────────────────────────┐
│  OpenClaw Studio (Header)               │
├──────────┬──────────────────────────────┤
│          │                              │
│ 需求中心 │  主要内容区域                │
│ 知识库   │  (案例列表/详情/视图)        │
│          │                              │
│ (侧边栏) │                              │
└──────────┴──────────────────────────────┘
```

## 📍 导航指引

### 左侧导航栏

#### 需求中心 (/cases)
- **图标**: 📄 文档图标
- **功能**: 查看和管理所有案例
- **操作**: 点击进入需求中心

#### 知识库 (/knowledge)
- **图标**: 📚 书籍图标
- **功能**: 浏览和搜索知识库内容
- **操作**: 点击进入知识库

### 面包屑导航

每个视图顶部都有面包屑导航，显示当前位置：

```
需求中心 > 案例标题 > 当前视图
```

- **需求中心**: 点击返回需求中心
- **案例标题**: 显示当前案例标题
- **当前视图**: 显示当前视图名称（规划视图/执行视图/测试视图/历史视图）

## 🎨 各视图交互指引

### 需求中心视图

#### 主要操作
1. **创建案例**
   - 位置: 左上角 **"+ 创建案例"** 按钮
   - 操作: 点击打开创建表单
   - 填写: 标题、描述、仓库路径（可选）、分支（可选）
   - 提交: 点击"创建"按钮

2. **查看案例详情**
   - 位置: 案例列表中的 **"查看详情"** 链接
   - 操作: 点击进入案例详情页
   - 或: 点击案例标题

3. **更多操作**
   - 位置: 案例列表中的 **"更多"** 链接
   - 功能: 编辑、删除等操作（根据实现）

#### 状态标签说明
- **已创建** (灰色): 案例刚创建，未开始
- **规划中** (蓝色): 正在生成或已生成计划
- **执行中** (蓝色): 正在生成代码补丁
- **测试中** (橙色): 正在生成测试建议
- **已完成** (绿色): 案例已完成并归档

### 规划视图

#### 主要操作
1. **生成计划**
   - 位置: 右上角 **"生成计划"** 按钮
   - 操作: 点击触发计划生成
   - 等待: 10-30 秒（显示加载状态）
   - 结果: 显示生成的计划和任务列表

2. **查看计划**
   - 标签: **"计划"** 标签
   - 内容: Markdown 格式的实现计划
   - 编辑: 可以编辑计划内容（点击编辑按钮）
   - 保存: 点击"保存"按钮保存修改

3. **查看任务列表**
   - 标签: **"任务列表"** 标签
   - 内容: 所有任务的表格
   - 操作: 
     - 点击任务查看详情
     - 切换任务状态（pending/completed）
     - 查看任务关联的文件

#### 提示信息
- **暂无计划**: 显示提示，引导点击"生成计划"
- **加载中**: 显示加载动画和提示文字
- **错误提示**: 显示错误信息和重试按钮

### 执行视图

#### 主要操作
1. **生成补丁**
   - 位置: "生成补丁"卡片
   - 步骤:
     1. 从下拉框选择任务
     2. 点击 **"生成补丁"** 按钮
     3. 等待生成完成
   - 结果: 补丁出现在补丁列表中

2. **查看补丁**
   - 位置: "补丁详情"卡片
   - 操作: 点击补丁的 **"查看"** 按钮
   - 功能: 在 Modal 中预览代码差异（Diff）

3. **应用补丁**
   - 位置: 补丁列表中的 **"应用补丁"** 按钮
   - 操作: 点击并确认
   - 结果: 补丁状态更新为"已应用"

4. **筛选和排序**
   - 位置: "补丁列表"卡片
   - 筛选:
     - 按文件路径搜索
     - 按任务 ID 筛选
   - 排序:
     - 按任务 ID 或生成时间
     - 升序/降序切换

#### 提示信息
- **暂无补丁**: 显示提示，引导生成补丁
- **选择任务提示**: 如果没有任务，提示先生成计划

### 测试视图

#### 主要操作
1. **生成测试建议**
   - 位置: 右上角 **"生成测试建议"** 或 **"重新生成"** 按钮
   - 操作: 点击触发测试分析
   - 等待: 10-30 秒
   - 结果: 显示潜在问题、测试用例、验收清单

2. **查看潜在问题**
   - 位置: "潜在问题"部分
   - 筛选: 按严重程度筛选（全部/严重/高/中/低）
   - 内容: 问题描述、关联文件

3. **查看测试用例**
   - 位置: "测试用例"部分
   - 操作: 点击展开查看详情
   - 内容: 测试步骤、预期结果

4. **管理验收清单**
   - 位置: "验收清单"部分
   - 操作: 勾选已完成的验收项
   - 保存: 点击 **"保存验收状态"** 按钮
   - 进度: 查看完成进度百分比

#### 提示信息
- **暂无测试结果**: 显示提示，引导生成测试建议
- **加载中**: 显示加载动画

### 历史视图

#### 主要操作
1. **查看时间线**
   - 位置: 主要内容区域
   - 内容: 按时间顺序显示所有开发事件
   - 类型: planning, coding, testing 等

2. **筛选历史记录**
   - 位置: 顶部筛选栏
   - 筛选:
     - 按类型筛选（多选）
     - 按时间范围筛选
     - 按关键词搜索

3. **查看统计信息**
   - 位置: 顶部统计卡片
   - 内容: 总事件数、各类型事件数量

4. **查看详情**
   - 操作: 点击历史记录项
   - 内容: 显示详细的事件信息

## 🔔 交互提示说明

### 成功提示
- **绿色消息**: 操作成功（如"案例创建成功"）
- **位置**: 页面右上角
- **自动消失**: 3 秒后自动消失

### 错误提示
- **红色消息**: 操作失败（如"生成失败"）
- **位置**: 页面右上角
- **内容**: 显示错误原因

### 警告提示
- **橙色消息**: 需要注意的信息
- **位置**: 页面内 Alert 组件
- **示例**: "暂无计划，请点击生成计划"

### 加载状态
- **Spin 组件**: 显示加载动画
- **位置**: 操作按钮或内容区域
- **文字**: "加载中..." 或具体操作描述

## 🎯 操作流程指引

### 新用户首次使用流程

```
1. 访问需求中心
   ↓
2. 创建第一个案例
   ↓
3. 进入规划视图
   ↓
4. 生成计划（等待完成）
   ↓
5. 查看任务列表
   ↓
6. 进入执行视图
   ↓
7. 选择任务生成补丁
   ↓
8. 查看补丁内容
   ↓
9. 进入测试视图
   ↓
10. 生成测试建议
   ↓
11. 查看测试结果
   ↓
12. 进入历史视图
   ↓
13. 查看开发历史
```

### 继续开发已有案例流程

```
1. 在需求中心选择案例
   ↓
2. 查看当前状态
   ↓
3. 根据状态进入相应视图
   - 无计划 → 规划视图
   - 有计划无补丁 → 执行视图
   - 有补丁无测试 → 测试视图
   ↓
4. 执行相应操作
   ↓
5. 查看历史记录
```

## 💡 使用技巧

### 1. 快速导航
- 使用左侧导航栏快速切换主要功能
- 使用面包屑导航返回上级页面
- 使用浏览器前进/后退按钮

### 2. 批量操作
- 在执行视图中可以生成多个补丁
- 在测试视图中可以批量勾选验收项

### 3. 数据查看
- 使用筛选和排序功能快速找到需要的内容
- 使用搜索功能查找关键词

### 4. 错误处理
- 遇到错误时查看错误提示信息
- 使用"重试"按钮重新尝试操作
- 检查浏览器控制台的错误信息

## 📱 响应式设计

### 桌面端（推荐）
- 最佳体验在 1920x1080 或更大屏幕
- 侧边栏和内容区域并排显示

### 平板端
- 侧边栏可能折叠
- 内容区域自适应宽度

### 移动端（部分支持）
- 侧边栏自动折叠
- 部分功能可能需要优化

## 🎨 视觉指引

### 颜色含义
- **蓝色**: 主要操作按钮、链接
- **绿色**: 成功状态、已完成
- **橙色**: 警告、进行中
- **红色**: 错误、失败
- **灰色**: 默认状态、已创建

### 图标含义
- **📄**: 文档、案例
- **📚**: 知识库
- **🔄**: 刷新、重新生成
- **➕**: 创建、添加
- **👁️**: 查看、详情
- **✏️**: 编辑
- **🗑️**: 删除
- **✅**: 完成、通过
- **⚠️**: 警告、注意
