# OpenClaw Studio 快速开始指南

## 🎯 5 分钟快速体验

### 步骤 1: 启动服务（1 分钟）

```bash
# 终端 1: 启动后端
python start_backend.py

# 终端 2: 启动前端
cd openclaw-studio-frontend
npm run dev
```

### 步骤 2: 创建案例（1 分钟）

1. 打开浏览器：http://localhost:5173
2. 点击 **"+ 创建案例"** 按钮
3. 填写信息：
   - **标题**：实现用户登录功能
   - **描述**：添加用户登录功能，包括登录表单、验证逻辑、会话管理
4. 点击 **"创建"**

### 步骤 3: 生成计划（1-2 分钟）

1. 在案例列表中点击刚创建的案例
2. 进入 **"规划视图"**
3. 点击右上角 **"生成计划"** 按钮
4. 等待生成完成（10-30 秒）
5. 查看生成的任务列表

### 步骤 4: 生成代码（1 分钟）

1. 切换到 **"执行视图"**
2. 在"生成补丁"区域选择一个任务
3. 点击 **"生成补丁"** 按钮
4. 等待生成完成
5. 点击 **"查看"** 预览代码差异

### 步骤 5: 生成测试（1 分钟）

1. 切换到 **"测试视图"**
2. 点击 **"生成测试建议"** 按钮
3. 等待生成完成
4. 查看潜在问题和测试用例
5. 勾选验收清单中的已完成项
